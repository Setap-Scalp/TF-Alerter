# TF-Alerter ⌚

Профессиональное бесплатное open-source desktop-приложение для трейдеров с уведомлениями о переходах таймфреймов, фандингах, листингах и торговых сессиях.

## 🧭 Что это за программа

TF-Alerter помогает не пропускать важные рыночные события в течение дня.
Программа отслеживает данные в реальном времени, показывает информацию в удобном интерфейсе и озвучивает события через TTS.

## ✨ Возможности

- ⏱️ Уведомления о переходах таймфреймов (с учетом времени биржи)
- 💸 Мониторинг фандинга по поддерживаемым биржам
- 📈 Мониторинг листингов и логирование событий
- 🌍 Мониторинг торговых сессий с обратным отсчетом до следующей сессии
- 🗣️ Гибкая настройка TTS (движок, язык, голос)
- 🕐 Overlay-часы с режимами отображения по окнам
- 🎛️ Настройки интерфейса: масштаб, цвета, шрифт, горячая клавиша
- 🧾 Журналы событий для оперативного контроля

## 🔓 Open Source

Проект открыт: исходный код доступен всем.
Любой может изучить архитектуру, проверить логику работы, предложить улучшения и внести вклад в развитие.

## 📥 Как установить

1. Перейдите в раздел **Releases** на GitHub: https://github.com/SetapScalp/TF-Alerter/releases/tag/V1.3
2. Загрузите TF-Alerter.7z
3. Распакуйте архив в любую папку.
4. Запустите **TF-Alerter.exe** и можно пользоваться. 


## 🧑‍💻 Запуск из исходников (VS Code)

Если вы скачали код из репозитория и хотите запускать через VS Code:

1. Откройте папку проекта в VS Code
2. Создайте окружение и установите зависимости:

```bash
python -m venv .venv
.venv\Scripts\python.exe -m pip install --upgrade pip
.venv\Scripts\python.exe -m pip install -r requirements.txt
```

3. Запуск:

```bash
.venv\Scripts\python.exe main.py
```

Или просто через `run.bat` (он сам ищет `.venv`).

> В репозиторий специально не добавляются `.venv`, `build`, `dist`, `__pycache__` — они большие/временные и должны создаваться на каждой машине локально.

## 💻 Системные требования

- ОС: Windows 10/11
- Процессор: 2+ ядра, от 1.8 ГГц.
- ОЗУ: минимум 4 ГБ
- Диск: от 300 МБ свободно (программа + кэш + временные TTS-файлы).
- Аудио: любое рабочее устройство вывода (динамики/наушники) для звуков и TTS.
- Интернет: обязателен для получения данных бирж для фандингов, листингов и для озвучки Edge TTS; без интернета мониторинг и онлайн-озвучка не работают.
