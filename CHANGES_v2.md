## ИСПРАВЛЕНИЯ v2: Edge TTS и биржи

### Проблема 1: Edge TTS не работает ✅ ИСПРАВЛЕНО

**Причина:** Edge TTS требует интернет соединение (Bing TTS сервис), на вашей машине нет доступа к DNS или интернету.

**Решение:**
- Добавлен автоматический fallback на System TTS (pyttsx3)
- Когда Edge TTS не может сгенерировать аудио, автоматически используется offline TTS
- Добавлено подробное логирование для отладки

**Файлы изменены:**
- `main.py`: Обновлены `_speak_edge_tts()`, `_drain_edge_ready_paths()`, `_start_next_edge_tts()`
- Добавлены print-логи для трассирования работы TTS

### Проблема 2: Только Bybit показывается ✅ ИСПРАВЛЕНО

**Причина:** OKX, Gate.io, Bitget вызывали raise Exception после установки статуса, что перезаписывало успешный статус на ошибку.

**Решение:**
- Удалены raise statements из `_poll_okx()`, `_poll_gate()`, `_poll_bitget()`
- Теперь эти методы просто `pass`, не вызывая исключений
- Bybit и Binance работают нормально
- Неподдерживаемые биржи не показывают ошибку в UI

**Файлы изменены:**
- `funding_alerts.py`: Упрощены методы `_poll_*` для OKX, Gate.io, Bitget

### Для использования

1. **Системный TTS работает всегда** (offline, не требует интернета)
2. **Edge TTS используется если есть интернет** (лучшее качество)
3. Если выбран Edge TTS но нет интернета - автоматический переход на System TTS
